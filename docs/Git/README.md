## 专题讨论

### Git 是什么？

`Git`是 Linus(没错，又是他)开源的分布式版本控制工具。

### 版本控制工具有哪些？

1. 集中型：`svn`、`cvs`、`vss`等，对中央仓库依赖严重，且上传都是完整版，不好溯源，不易多版本协同
2. 分布式：`BitKeeper`、`Git`等
