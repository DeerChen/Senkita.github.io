1.2. 标准流程图

```flow
    st=>start: 开始框
    op=>operation: 处理框
    cond=>condition: 判断框
    sub1=>subroutine: 子流程
    io=>inputoutput: 输入输出框
    e=>end: 结束框

    st(right)->op->cond
    cond(true)->io->e
    cond(false)->sub1(right)->op
```
